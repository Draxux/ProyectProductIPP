Serie Fibonacci
----------------
Funcionamiento
Se abre el archivo 'index.php' que es la pagina inicial.
 
1. Al abrir la página, se muestra un título "Serie Fibonacci".

2. En la página, hay un campo de entrada donde los usuarios pueden ingresar un número de la serie Fibonacci que desean generar. El número debe estar entre 1 y 99.

3. Después de ingresar el número deseado, el usuario puede hacer clic en el botón "Generar" para generar la secuencia de Fibonacci correspondiente.

4. Una vez que el usuario hace clic en el botón, el formulario se envía utilizando el método POST de HTML y se procesa en el lado del servidor utilizando PHP.

5. El servidor PHP valida el número ingresado y genera la secuencia de Fibonacci correspondiente utilizando una función.

6. La secuencia de Fibonacci generada se muestra en la página debajo del botón "Generar", dentro de un contenedor con el id "fibonacci-sequence".
